  • Migration to Cloud Computing

    summary: This master thesis is in the Cloud Computing domain, one that is growing currently. Migrate legacy application to the Cloud and migrate from one Cloud provider to another one pose some challenges, namely regarding state management as Cloud architectures rely on distributed Cloud systems. The goals of this master thesis work are: 1) To analyse and compare the existing models of Cloud Computing migration and state management models in distributed Cloud systems, 2) To develop a Cloud Computing innovative migration methodology that addresses the deficiencies found in the existing models and 3) To implement it as software prototype and to test it.
